# Health Data Analysis with Python

Exploratory analysis of pharmaceutical consumption data using Python.

This project demonstrates a simple data analysis workflow using Python, including data loading, aggregation, and visualization.

## Dataset

The dataset included in this repository is a small synthetic sample inspired by public health data on antibiotic consumption in France.

Columns in the dataset:

- **year** : year of observation
- **region** : French region
- **drug_class** : therapeutic class of the drug
- **boxes** : number of boxes dispensed

## Objectives

The goal of this project is to:

- load and inspect a dataset
- perform basic data aggregation
- visualize trends in drug consumption
- generate graphical outputs

## Technologies used

- Python
- pandas
- matplotlib

## Analysis performed

Two analyses are performed:

1. **Antibiotic consumption by year**

This visualization shows the evolution of antibiotic consumption over time.

2. **Antibiotic consumption by region**

This chart compares total antibiotic consumption across regions.
